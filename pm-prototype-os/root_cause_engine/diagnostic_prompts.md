# Diagnostic Prompts

Templates for moving from an initial request to likely causes and next validation steps.

## Flow

```text
statement
request
possible causes
clarifying prompt
next hypothesis
```

## Prompt Types

### Moment

```text
Where in the workflow does the issue become most painful?
```

### Frequency

```text
How often does this occur?
```

### Impact

```text
Does this mainly affect time, quality, rework, cost, or decision speed?
```

### Input

```text
What files, records, or data are used for this work?
```

### Owner

```text
Who makes the final judgment?
```

### Current Operation

```text
How is this handled today?
```

### Success Condition

```text
What would need to appear in the first demo for this to feel valuable?
```

## Output Template

```text
## Statement

## Surface Request

## Hypothesis A

## Hypothesis B

## Hypothesis C

## First Prompt

## What It Separates

## Next Action By Answer
- If answer 1:
- If answer 2:
- If answer 3:
```
