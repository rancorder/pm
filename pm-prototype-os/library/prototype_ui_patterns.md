# Prototype UI Patterns

PM Prototype OSで、顧客確認用プロトタイプを高速生成するためのUIパターン集。

## 基本思想

プロトタイプUIは美しさより、業務フローと判断ポイントを伝える。

```text
入力
↓
AI解析
↓
人間レビュー
↓
出力
↓
次アクション
```

---

## UI Pattern 01: File Analysis Dashboard

### 用途

- 図面解析
- 仕様書解析
- IFCチェック
- 見積抽出

### 画面構成

- ファイルアップロード
- 解析ステータス
- 抽出結果一覧
- 不備候補
- レポート出力

### 顧客に刺さる見せ方

「今まで人が探していた確認候補を、一覧で先に出す」

## Codex / v0 / Lovable 用プロンプト

```text
以下のUIパターンに沿って、顧客確認用のWebプロトタイプを作成してください。

# UIパターン

# 業務目的

# 入力データ

# AI解析結果

# 人間が確認する項目

# 出力

# 制約
- ログイン不要
- DB不要
- ダミーデータでよい
- 業務システム風の簡潔なUI
- 顧客が価値判断できることを優先
```
