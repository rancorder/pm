# Tech Card: BIM AI Agent

## 技術名

BIM AI Agent / 自然言語によるBIM操作支援

## 参照候補

- Lightning BIM AI Agent周辺思想
- Text2BIM系研究
- Revit Add-in OSS
- AI Agent framework

## 何ができるか

設計者やPMが自然言語で指示し、BIM/Revit操作・チェック・レポート生成・作業手順化を支援する。

## 解決できる顧客課題

- Revit操作が属人化している
- 熟練者でないと作業できない
- 繰り返し操作が多い
- 過去の操作手順が蓄積されない
- PMが技術作業の粒度を理解しづらい

## Arent領域との親和性

- BIM: 最高
- Revit: 最高
- IFC: 高
- 配筋: 中〜高
- プラント: 中
- 図面/仕様書RAG: 高

## 入力

- 自然言語指示
- BIMモデル
- Revit環境
- 操作履歴
- 設計ルール

## 出力

- 実行予定操作
- 操作手順
- 自動実行結果
- エラー候補
- 再利用可能なコマンド履歴

## MVPでの使い方

自然言語で「このモデルの属性欠損を確認して」などと入力すると、実行手順と想定結果を表示する。初期MVPでは実操作せず、操作計画の提示だけでも価値検証できる。

## プロトタイプ案

```text
Natural Language Command
↓
Intent Classification
↓
Operation Plan
↓
Risk/Confirmation
↓
Execution Log Mock
```

## 実装難易度

中〜高

## 注意点

- 実行権限を持たせる場合は危険操作のガードレールが必要
- PoCでは「実行」より「操作計画提示」から始める方が安全
- 顧客ごとのRevit運用ルール差分が大きい

2. 操作ミスが起きると影響が大きい作業は何か？
